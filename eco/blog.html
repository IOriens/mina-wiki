      # 小程序博客搭建工具对比

      # mini-blog (opens new window)

      项目地址：https://github.com/CavinCao/mini-blog (opens new window)

      mini-blog是一款基于云开发的博客小程序，该小程序完全不依赖任何后端服务，无需自己的网站、服务器、域名等资源，只需要自行注册小程序账号即可。

      # WeHalo (opens new window)

      项目地址：https://github.com/aquanlerou/WeHalo (opens new window)

      WeHalo [wiˈheɪloʊ]，意为我们的光环，嘻嘻。

      配合 Halo (opens new window) 轻快，简洁，功能强大的博客系统而开发出来的 简约风 微信小程序版博客
